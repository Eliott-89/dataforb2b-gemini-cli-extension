# DataForB2B — Gemini CLI Extension

Power your Gemini CLI with live B2B data. Search people and companies, enrich profiles, find verified emails — all from your terminal.

## Install

gemini extensions install https://github.com/Eliott-89/dataforb2b-gemini-cli-extension

Gemini CLI will open a browser to authenticate with your DataForB2B account.

## Tools

- search_natural_language: Find people or companies using plain English
- enrich_profile: Enrich a LinkedIn profile with email + full data
- enrich_company: Get company info from a domain or LinkedIn URL
- search_lookalike: Find similar companies or people
- find_email: Find a verified work email from name + company
